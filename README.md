Tipos de datos
NUMERO (números enteros) (0-9)
TEXTO (cadenas entre comillas) (Aa-Zz)

Palabras reservadas
MOSTRAR (imprimir en pantalla)
CAPTURAR (leer entrada del usuario)
entero (declarar variable numérica)
texto (declarar variable de cadena)

Operadores permitidos
+ SUMA n+n
- RESTA n-n
* MULTIPLICACION n*n
/ DIVISION n/n (Verificar ≠0)

Declaración
expresion = NUMERO | OPERADOR | NUMERO | ;
[NUMERO|TEXTO] nombre_variable ;
nombre_variable = [valor|expresión] ;
mostrar [valor|expresión|variable] ;

Mostrar en pantalla
MOSTRAR = [NUMERO | "TEXTO" | VARIABLE]

Sintaxis obligatoria
; al final de cada instrucción
declarar antes de usar: EJEMPLO: entero edad1 ->  edad1 = 16;
No redeclarar variables
Tipos compatibles: NUMERO + NUMERO, TEXTO + TEXTO

Verificaciones
Variable declarada antes de uso
Sin redeclaraciones
Compatibilidad de tipos
Operaciones entre tipos iguales
No división por cero
